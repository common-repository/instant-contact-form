== Description ==
A complex contact form is great when you need it, but sometimes you just want something simple. This is where Instant Contact Form comes in. It sends all messages to the site owner's email address (or a custom address) and stores the message in a database. Includes fields: First Name, Last Name, Email, Subject, and Message. Built-in CAPTCHA. No learning curve. Install, activate, place the shortcode, done. 

== Installation ==
1. Go to WordPress Dashboard -> Plugins -> Add New
2. Search for Instant Contact Form
3. Find the plugin and click the Install Now button
4. Activate the plugin
5. Shortcode: [instant_contact_form]
